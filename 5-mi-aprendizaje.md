### Mi aprendizaje
<br>docker pull descargar imagenes
<br>docker run , resume docker pull , docker create, docker start
<br>docker run  -d --name nombre -p puertos imagen
<br>docker rm remover contenedor
<br>docker rmi remover imagen 
<br>docker exec -it o -i  bash sirven para dejar una terminal abierta para interactuar dentro del contenedor
<br>docker rm -f es para forzar la eliminacion de un contenedor en ejecucion ( no recomendable)
<br>tener cuidado con la asignacion de puertos y revisar bien la documentacion de las imagenes para su respectivo mapeo de puertos 
<br>docker ps -a mostrar contenedores en o no ejecucion Xd
